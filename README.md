# CS50 Practice Assignments

This repository contains multiple C exercises.

---

## Part 1 — Iteration
File: iteration.c

Use loops to solve the problem.
Validate input properly.

---

## Part 2 — Recursion
File: recursion.c

Solve the problem using recursion.
At least one recursive function must be used.

---

## Part 3 — Searching
Files:
- search0.c (integers)
- search1.c (strings)

Print exactly:
Found
or
Not found

---

## Part 4 — Phonebook
Files:
- phonebook0.c
- phonebook1.c

Modify the program to allow:
- Adding a new contact
- Searching for a contact

---

## Rules
- Do not rename files.
- Compile with:
  make filename
- Must work on cs50.dev.
